
# 🏥 Hospital Appointment and Record Management System

## 📌 Overview

The **Hospital Appointment and Record Management System** is a backend-based application developed using Spring Boot. It is designed to manage hospital operations such as patient appointments, doctor schedules, and medical records through RESTful APIs.

This system focuses on efficient data handling, secure access, and structured management of healthcare information.

---

## 🎯 Features

* 📅 **Appointment Management**

  * Create, update, and cancel appointments
  * Manage doctor availability

* 👨‍⚕️ **Doctor Management**

  * Add and manage doctor details
  * Maintain specialization and schedules

* 👩‍💼 **Patient Management**

  * Register and manage patient information
  * Maintain patient history

* 📋 **Medical Records**

  * Store prescriptions and diagnosis
  * Retrieve complete medical records

* 🔐 **Authentication & Security**

  * Role-based access (Admin, Doctor, Patient)

---

## 🛠️ Tech Stack

* **Backend:** Java, Spring Boot
* **Database:** MySQL
* **API Testing:** Postman
* **Version Control:** Git & GitHub

---

## 🏗️ Project Structure

* **Controller Layer** → Handles HTTP requests and APIs
* **Service Layer** → Contains business logic
* **Repository Layer** → Manages database interactions using JPA
* **Entity Layer** → Defines database schema

   git clone https://github.com/your-username/your-repo-name.git

4. Run the Spring Boot application

5. Test APIs using Postman

---

## 📊 API Workflow

1. Admin/Patient creates account
2. Patient books appointment
3. Doctor views scheduled appointments
4. Doctor updates diagnosis & prescription
5. Data is stored and managed in database
